🚀 КОМАНДЫ ДЛЯ ПРИМЕНЕНИЯ:

1. Применить миграцию БД:
   
   sqlite3 school_canteen.db < add-notifications-table.sql

2. Перезапустить Docker:
   
   docker-compose -f docker-compose.production.yml down
   docker-compose -f docker-compose.production.yml up -d --build

3. Проверить логи:
   
   docker-compose -f docker-compose.production.yml logs -f backend

✅ БЫСТРАЯ ПРОВЕРКА:

ТЕСТ 1: Числовые коды (2 минуты)
  → Войти как студент
  → Создать заказ
  → Нажать "🔢 Показать код"
  → Увидеть 6-значный код (например: 456789)
  → Войти как повар
  → Нажать "🔢 Ввести код"
  → Ввести код
  → Выдать заказ

ТЕСТ 2: Предложение блюда (1 минута)
  → Войти как повар
  → "Новые блюда" → "+ Предложить блюдо"
  → Заполнить форму
  → Отправить
  → Проверить, что нет ошибок

ТЕСТ 3: Уведомления (2 минуты)
  → Войти как повар
  → Создать заявку на закупку
  → Войти как администратор
  → Одобрить заявку
  → Войти как повар
  → Увидеть уведомление (розовая карточка)
  → Открыть и прочитать
