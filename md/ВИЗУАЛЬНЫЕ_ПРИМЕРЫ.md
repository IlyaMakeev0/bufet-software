## 1️⃣ МОДАЛЬНОЕ ОКНО ПОДПИСКИ

### ДО исправлений ❌
```
┌─────────────────────────────────────┐
│ Оформление подписки                 │ ← Белый текст на белом фоне
├─────────────────────────────────────┤
│ Выберите тип подписки:              │ ← Не видно
│                                     │
│ ┌──────────┐  ┌──────────┐         │
│ │ ЗАВТРАК  │  │   ОБЕД   │         │ ← Белый текст
│ │          │  │          │         │
│ └──────────┘  └──────────┘         │
│                                     │
│ ☐ Понедельник                       │ ← Белый текст на белом
│ ☐ Вторник                           │
└─────────────────────────────────────┘
```

### ПОСЛЕ исправлений ✅
```
┌─────────────────────────────────────┐
│ Оформление подписки                 │ ← ТЕМНЫЙ текст (#2c3e50)

│                                     │
│ ┌──────────┐  ┌──────────┐         │
│ │ ЗАВТРАК  │  │   ОБЕД   │         │ ← ТЕМНЫЙ текст
│ │ Завтраки │  │ Обеды    │         │ ← Серый (#546e7a)
│ └──────────┘  └──────────┘         │

│ ☑ Вторник                           │
```
