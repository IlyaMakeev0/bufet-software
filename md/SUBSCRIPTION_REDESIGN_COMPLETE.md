# Редизайн модального окна абонемента ✓

## Дата: 27 января 2026

## Задача
Переделать дизайн модального окна оформления абонемента - сделать его более горизонтальным и в общем стиле приложения.

---

## Реализованные изменения

### 1. Горизонтальная раскладка

**Типы абонементов (2 колонки):**
```
┌─────────────────┐  ┌─────────────────┐
│   BREAKFAST     │  │   FULL DAY      │
│ Только завтрак  │  │  Полный день    │
│ Завтрак каждый  │  │ Завтрак + Обед  │
│      день       │  │   + Полдник     │
└─────────────────┘  └─────────────────┘
```

**Длительность абонемента (3 колонки):**
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ 7 ДНЕЙ   │  │ 14 ДНЕЙ  │  │ 30 ДНЕЙ  │
│  700 ₽   │  │ 1300 ₽   │  │ 2500 ₽   │
│100₽/ДЕНЬ │  │ 93₽/ДЕНЬ │  │ 83₽/ДЕНЬ │
│          │  │ ВЫГОДНО  │  │ЛУЧШАЯ    │
│          │  │          │  │  ЦЕНА    │
└──────────┘  └──────────┘  └──────────┘
```

### 2. Увеличенная ширина модального окна
- Было: 640px
- Стало: 900px
- Больше пространства для горизонтальной раскладки

### 3. Улучшенный дизайн карточек

**Размеры и отступы:**
- Padding: 24px (было 20px)
- Border-radius: 12px (было 8px)
- Min-height для типов: 140px
- Gap между карточками: 20px для типов, 16px для длительности

**Границы:**
- Толщина: 3px (более заметные)
- Цвет: #bdc3c7 (светло-серый)
- При hover: #1565c0 (синий)

**Эффекты:**
- Transform при hover: translateY(-4px) - больше подъём
- Тени: более выраженные (0 8px 24px)
- Плавные переходы: 0.3s
