⚡ БЫСТРЫЙ СТАРТ:

1. Загрузите код на GitHub:
   git add .
   git commit -m "Prepare for Render"
   git push origin main

2. Откройте Render Dashboard:
   https://dashboard.render.com

3. Создайте PostgreSQL базу:
   New + → PostgreSQL
   Name: school-canteen-db
   Plan: Free

4. Создайте Web Service:
   New + → Web Service
   Name: school-canteen-app
   Build: npm install && npm run build
   Start: npm start

5. Настройте переменные окружения:
   - NODE_ENV=production
   - PORT=10000
   - DB_HOST, DB_PORT, DB_NAME, DB_USER, DB_PASSWORD
   - DATABASE_URL
   - SESSION_SECRET

6. Примените тестовые аккаунты:
   Shell → cat create-test-accounts.sql | psql $DATABASE_URL

7. Откройте ваш URL и войдите:
   student@test.com / test123
