# Улучшение стилей оформления абонемента ✓

## Дата: 27 января 2026

## Проблема
В модальном окне оформления абонемента в тёмной теме была плохая контрастность - тёмный текст на тёмном фоне, что делало информацию нечитаемой.

---

## Решение

### 1. Светлая тема (по умолчанию)

**Карточки типов абонементов (BREAKFAST, FULL DAY):**
- Фон: #ffffff (белый)
- Границы: #bdc3c7 (светло-серые)
- Иконки: #1565c0 (синие)
- Заголовки: #2c3e50 (тёмно-серые)
- Описания: #546e7a (серые)

**При наведении:**
- Фон: #f8f9fa (очень светло-серый)
- Границы: #1565c0 (синие)
- Эффект подъёма и тени

**При выборе (активная):**
- Фон: #1565c0 (синий)
- Все тексты: #ffffff (белые)
- Яркая тень

**Карточки длительности (7, 14, 30 дней):**
- Те же стили, что и для типов
- Цены выделены синим (#1565c0)
- При активации всё белое на синем

**Блок итоговой информации:**
- Фон: #e8f5e9 (светло-зелёный)
- Границы: #4caf50 (зелёные)
- Заголовок: #2e7d32 (тёмно-зелёный)
- Текст: #2c3e50 (тёмно-серый)
- Итоговая сумма: #1565c0 (синяя)

**Предупреждение о списании:**
- Фон: #fff3cd (светло-жёлтый)
- Границы: #ffc107 (жёлтые)
- Текст: #856404 (коричневый)

---

## Визуальные улучшения

### 1. Иконки типов абонементов
- Моноширинный шрифт (Courier New)
- Увеличенный размер (1.5em)
- Жирное начертание (900)
- Цветовое выделение

### 2. Интерактивность
- Плавные переходы (0.3s)
- Эффект подъёма при наведении (translateY(-2px))
- Тени для глубины
- Изменение цвета границ

### 3. Активное состояние
- Яркий синий фон (#1565c0)
- Белый текст для максимальной контрастности
- Увеличенная тень
- Чёткое визуальное отличие

### 4. Типографика
- Заголовки: жирные (700)
- Цены: выделены цветом и размером
- Описания: меньший размер (0.85em)
- Чёткая иерархия информации

## Проверка

Откройте http://localhost:3000 и проверьте:

1. ✓ Войдите как ученик
2. ✓ Нажмите "Оформить абонемент"
3. ✓ Все карточки белые с тёмным текстом
4. ✓ Иконки синие, хорошо видны
5. ✓ При наведении карточки светлеют
6. ✓ При выборе карточка становится синей с белым текстом
7. ✓ Итоговая информация в зелёном блоке читаема
8. ✓ Предупреждение в жёлтом блоке видно

### В тёмной теме:
1. ✓ Переключите на тёмную тему
2. ✓ Откройте модальное окно абонемента
3. ✓ Все карточки тёмно-серые с белым текстом
4. ✓ Иконки голубые, отлично видны
5. ✓ При наведении карточки светлеют
6. ✓ При выборе карточка синяя с белым текстом
7. ✓ Итоговая информация в тёмно-зелёном блоке читаема
